# FinPulse Lite

FinPulse Lite is a Python-based stock market data analysis project. It uses the yfinance library to download historical stock market data and stores it as CSV files for further analysis using Pandas.

Currently, the project downloads 5 years of daily data for 10 NIFTY 50 stocks including Reliance, TCS, Infosys, HDFC Bank, ICICI Bank, SBI, ITC, L&T, Hindustan Unilever, and Kotak Mahindra Bank. The data is saved in the data folder, with one CSV file per stock.
