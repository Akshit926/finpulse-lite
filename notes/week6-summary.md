### **1. Why is interactivity more valuable than static reports?**

Interactive dashboards allow users to explore data by changing inputs such as stocks, strategies, or parameters and immediately seeing the updated results. This makes analysis faster, supports better decision-making, and helps users compare different scenarios without generating multiple reports. Static reports only present fixed information, whereas interactive dashboards provide a more engaging and flexible experience.

---

### **2. What's the difference between Matplotlib and Plotly? When would you use each?**

**Matplotlib** is a static plotting library that is ideal for creating simple graphs, reports, and publication-quality figures. It is lightweight and widely used for data analysis.

**Plotly** creates interactive charts with features like zooming, panning, hovering, and dynamic tooltips. It integrates well with web applications such as Streamlit dashboards.

**When to use:**

* **Matplotlib:** Data analysis, research reports, quick visualizations, and exporting static images.
* **Plotly:** Interactive dashboards, web applications, and presentations where users need to explore the data.

---

### **3. What makes a dashboard "user-friendly" to a non-engineer?**

A user-friendly dashboard presents information in a simple and intuitive way without requiring technical knowledge. Important metrics should be clearly visible, charts should be easy to understand, and controls should have meaningful labels. Interactive filters, responsive layouts, and minimal clutter improve usability. Providing summaries, comparisons, and visual insights instead of raw data helps non-engineers quickly understand the results and make informed decisions.
